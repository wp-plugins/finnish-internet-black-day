== Description ==

"The day when Finnish Internet turns black." This plugin provides easy way to participate Finnish Internet black day-event on 23.4.2013.

Support[ Common sense into copyright law proposal](http://jarkea.fi/en) by using this plugin and showing your visitors pop-up message telling about proposal.

You can install and activate plugin right now, code will be active only on campaign day (23.4.). For pop-up message, this plugin uses jQuery code made by SC5.

[Demo](http://sc5.github.com/copyright-campaign) | [Original code](https://github.com/SC5/copyright-campaign)

== Installation ==

= Using The WordPress Dashboard =
1. Navigate to the 'Add New' Plugin Dashboard
2. Select blackday.zip from your computer
3. Upload
4. Activate the plugin on the WordPress Plugin Dashboard

= Using FTP =
1. Extract blackday.zip to your computer
2. Upload the blackday directory to your wp-content/plugins directory
3. Activate the plugin on the WordPress Plugins dashboard
